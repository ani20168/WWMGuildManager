# WWM Guild Manager

《燕雲十六聲》百業管理輔助工具。透過影像辨識模型辨識遊戲畫面，協助自動尋訪成員與智慧踢人，減少重複操作。

> [!WARNING]
> 本工具僅供個人輔助使用。
>
> 遊戲內容皆透過影像辨識識別，不透過遊戲記憶體，理論上不太可能被ban，但不保證不會有誤ban的風險，使用此工具視同您願意承擔此風險。
>
> 使用此工具造成的任何損失(包含但不限於帳號誤ban、百業踢錯人)，開發者不負任何責任。

---

## 系統需求

- Windows 10 / 11（64 位元）
- 已安裝並執行《燕雲十六聲》
- 建議使用 NVIDIA GPU (A卡不確定能不能跑)，VRAM 4GB以上

---

## 安裝與啟動


1. 前往 [Releases](https://github.com/ani20168/WWMGuildManager/releases) 下載最新版 `WWMGuildManager-v*-windows.zip`
2. 解壓縮後進入 `WWMGuildManager` 資料夾
3. 執行 `WWMGuildManager.exe` (建議以管理員身分執行，否則程式可能無法操作鍵盤及滑鼠)

## 使用方式

### 1. 首頁 — 基本設定

在開始使用前，建議先確認以下項目：

| 項目 | 說明 |
|------|------|
| 遊戲視窗 | 顯示是否已找到 `wwm.exe` |
| 快捷鍵 | 預設 **F8**，按一下開始識別、再按一下停止（可在首頁更改） |
| 識別延遲 | 每次掃描遊戲畫面的間隔（秒） |
| HDR 補償 | 若使用 HDR 螢幕且辨識不準，可開啟此選項 |
| 識別區域調整 | 視覺化調整 OCR 辨識框位置（需遊戲已啟動） |
| OCR 測試 | 用內建測試圖片確認 EasyOCR 是否正常 |

設定會自動儲存至 `config.json`。

---

### 2. 自動尋訪成員

在遊戲中開啟「尋訪成員」列表後使用此功能。

![自動尋訪成員](https://github.com/ani20168/WWMGuildManager/blob/main/images/for_readme/member_visit.png)

**操作步驟：**

1. 設定篩選條件（可選）：
   - **最低等級**：留空表示不限
   - **常用語言**：例如 `中文`，留空表示不限
2. 點擊 **▶ 開始識別**，或按全域快捷鍵 **F8**
3. 程式會自動 OCR 辨識玩家列表，對符合條件的玩家點擊「+」邀請
4. 若無符合玩家，會自動按 **R** 刷新列表（最短間隔 3 秒）
5. 再次按 **F8** 或點 **■ 停止識別** 結束

> [!NOTE]
> 若辨識位置偏移，可至首頁 → **調整尋訪清單識別區域** 進行校正。

---

### 3. 智慧踢人

在遊戲中開啟「百業成員」列表後使用此功能。

![智慧踢人](https://github.com/ani20168/WWMGuildManager/blob/main/images/for_readme/member_kick.png)

**使用前準備（重要）：**

1. 在遊戲中將成員列表依 **「本週活躍度」由低到高** 排序，確保貢獻最低的成員排在最上方
2. 點擊 **定義頭像位置**，倒數後將滑鼠移至**第一個成員的頭像**上，程式會記錄座標

**操作步驟：**

1. 設定 **貢獻度小於** 閾值（例如 `500`），低於此值的成員會被標記為符合踢出
2. （可選）設定 **職位白名單**，以逗號分隔（例如 `請假,半步演員`），這些職位即使貢獻不足也不會被踢
3. 點擊 **▶ 開始識別**，或按 **F8**
4. 程式會辨識列表最上方成員，若符合條件則自動點擊頭像 → 踢出百業 → 確認
5. 重複直到最上方成員活躍度高於閾值，或手動停止

 > [!NOTE]
 > 若 OCR 欄位對不準，可至首頁 → **調整踢人識別區域** 進行校正；**踢出確認延遲** 可在首頁的延遲設定中調整。

## 常見問題

**找不到遊戲視窗？**  
請確認遊戲已啟動且程序名稱為 `wwm.exe`，並以視窗模式或無邊框視窗執行。

**OCR 辨識不準？**  
先執行首頁的 OCR 測試；若使用 HDR 螢幕請開啟 HDR 補償；必要時調整識別區域。

**有識別到畫面，但是程式沒有動作？**  
EX:尋訪到成員，但是沒有把滑鼠移動到+號的情況

EX2:尋訪後沒有按R刷新名單

請以管理員身分執行程式。

**有支援其他語言嗎?**  
尋訪成員支持中英文

智慧踢人預設情況下僅支援繁體中文

如遊戲使用其他語言，需要自行截圖更換images內的圖片
